Виды документов

Как создать свой тип документа и настроить его поля

Простыми словами. Вид документа — это шаблон. Например: «Служебная записка», «Заявка на договор», «Заявка на пропуск». Вы сами решаете, какие типы документов есть в системе и какие данные в них заполняются.

У каждого вида есть стандартные реквизиты (они всегда) и ваши собственные поля (добавляете под свои нужды).

1Стандартные реквизиты

Эти поля есть у любого документа — их не нужно настраивать:

Номер

Присваивается автоматически по префиксу вида.

Дата

Дата документа.

Организация и Проект

Обязательные реквизиты.

Тема и Содержание

О чём документ.

Вложения

Файлы к документу.

Ваши поля

Всё остальное — добавляете сами (см. ниже).

2Собственные поля — конструктор

Здесь вы добавляете поля, которые нужны именно этому виду. Например, у «Заявки на договор» — поля Контрагент, Сумма, Срок.

3Типы полей

ТипДля чего
СтрокаКороткий текст в одну строку.
ТекстДлинный текст в несколько строк.
ЧислоОбычное число (количество, срок в днях).
ДеньгиСумма. Именно по такому полю удобно строить условия в матрице.
ДатаВыбор даты из календаря.
Да / НетГалочка (флажок).
Ссылка на справочникВыбор из списка: сотрудники, организации, проекты или свой справочник.

4Как создать вид документа — по шагам

  1. Нажмите «Новый вид» в правом верхнем углу.
  2. Впишите Название (например, «Заявка на договор») и Префикс для номеров (например, ДГ → номера будут ДГ-000001, ДГ-000002…).
  3. Нажмите «Поле» и добавьте нужное поле: укажите код (латиницей, например amount), название и тип.
  4. Отметьте галочкой «Обязательное», если поле нельзя оставлять пустым.
  5. Добавьте столько полей, сколько нужно, и нажмите «Сохранить».
💡 Про код поля Код — это техническое имя латиницей без пробелов (amount, contractor). Название — это то, что видит пользователь («Сумма»). Код менять после создания не стоит.

5Связь с матрицей согласования

Поля документа используются в условных этапах матрицы. Например, вы добавили денежное поле Сумма — теперь в матрице можно настроить: «этап Директора включается, только если Сумма > 1 000 000».

⚠️ Порядок действий Сначала создайте поле в виде документа (шаг 4), и только потом оно появится в списке полей при настройке условия в «Матрице согласования».

6Частые вопросы

Можно ли удалить вид документа?

Системный вид «Служебная записка» удалить нельзя. Остальные — можно, если по ним нет документов. Вместо удаления вид можно сделать неактивным.

Что такое «Ссылка на справочник»?

Это поле-выбор из готового списка: сотрудники, организации, проекты или ваш собственный справочник (раздел «Справочники»). Пользователь не вводит текст, а выбирает нужное значение.

Откуда берутся номера документов?

Система нумерует автоматически по префиксу вида. Счётчик у каждого вида свой.
